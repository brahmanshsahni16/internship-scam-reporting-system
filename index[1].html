About This Project DBMS

🗄️

DBMS Concepts Used

Primary & foreign keys, constraints, JOINs, GROUP BY, HAVING, subqueries, normalization (3NF), and transactional integrity.

🔗

Relational Mapping

6 tables: Users, Companies, Job_Postings, Reports, Verification, Reviews — all linked via foreign keys ensuring referential integrity.

🛡️

Safety Features

Scam pattern detection via report aggregation. Verification workflow for companies. Admin-controlled moderation dashboard.

⚙️

Tech Stack

MySQL for RDBMS · Node.js + Express for REST API · JWT authentication · Vanilla JS + CSS frontend · bcrypt password hashing.
